# GTM Strategy

## Target User

Teams that already spend money on multiple AI tools like ChatGPT, Claude, Cursor, Copilot, and Gemini.

Small teams do not actively track AI subscription costs, which creates overspending opportunities.

---

## When They Would Need This Tool

Users would likely search for this tool when:
- AI subscription costs start increasing
- teams begin scaling
- multiple employees start buying AI tools separately
